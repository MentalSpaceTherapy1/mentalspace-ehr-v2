## Interactive Elements

### Buttons
- **Primary Actions:** Contained buttons (blue)
- **Secondary Actions:** Outlined buttons
- **Destructive Actions:** Red contained buttons
- **Icon Buttons:** 48x48px minimum (touch-friendly)

### Forms
- **Auto-focus:** First input field on mount
- **Enter Key:** Submit form (when ready)
- **Validation:** Real-time with error messages
- **Loading States:** Disabled + spinner during submit

### Dialogs
- **Delete Confirmation:** Modal with Cancel/Delete
- **Detail View:** Sleep calendar day details
- **Edit Mode:** Pre-populated form

### Notifications
- **Success:** Green snackbar, bottom-right
- **Error:** Red snackbar, bottom-right
- **Auto-dismiss:** 6 seconds
- **Manual close:** X button

## Animation & Transitions

### Smooth Transitions
- Card hover: `transform: scale(1.02)`
- Button hover: Color shift

- FAB: Zoom in/out
- Confetti: 5-second duration

- Skeleton screens: Pulse animation
- Progress bars: Smooth fill
- Spinners: Circular rotation

## Accessibility Features

### Keyboard Navigation
- Tab through all interactive elements
- Enter to activate buttons
- Escape to close dialogs
- Arrow keys in dropdowns

### Screen Readers
- ARIA labels on all inputs
- Role attributes on custom components
- Alt text on icons
- Error announcements

### Visual Accessibility
- High contrast ratios (4.5:1 minimum)
- Focus indicators (blue outline)
- Error states (red with icon)
- Large tap targets (48x48px)
