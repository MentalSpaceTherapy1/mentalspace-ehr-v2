## Typography

### Headings
- **H4**: Page titles (bold, primary color)
- **H5**: Section headers
- **H6**: Card titles, subsections
- **Body1**: Regular text
- **Body2**: Secondary text
- **Caption**: Helper text, labels

### Font Weights
- **Bold (700)**: Totals, important numbers
- **Medium (500)**: Labels, card titles
- **Regular (400)**: Body text

## Icons Used

### Vendor Management
- `Business` - Vendor icon
- `AttachMoney` - Money/spend icon
- `Description` - Contract icon
- `Phone`, `Email`, `LocationOn` - Contact info
- `Upload` - W-9 upload
- `Edit`, `Visibility`, `Delete` - Actions

### Budget & Finance
- `AccountBalance` - Budget icon
- `Receipt` - Expense icon
- `ShoppingCart` - Purchase order icon
- `TrendingUp` - Charts/analytics
- `CalendarToday` - Dates
- `CheckCircle` - Approved status
- `Warning` - Warning status
- `Error` - Critical status
- `Download` - Export

## Status Colors

### Expense/PO Statuses
- **DRAFT**: Grey
- **PENDING**: Orange/Warning
- **APPROVED**: Green/Success
- **REJECTED/DENIED**: Red/Error
- **PAID/RECEIVED**: Blue/Info
- **CANCELLED**: Grey/Default

### Budget Health
- **HEALTHY**: Green (< 75% used)
- **WARNING**: Orange (75-90% used)
- **CRITICAL**: Red (> 90% used)

## Interactive Elements

### Hover Effects

transition: transform 0.2s, box-shadow 0.2s;

  box-shadow: 6;

### Button Styles
- **Primary**: Purple gradient
- **Success**: Green
- **Error**: Red
- **Outlined**: Border with hover background

## Responsive Breakpoints

- **xs**: < 600px (mobile)
- **sm**: 600px - 960px (tablet)
- **md**: 960px - 1280px (small desktop)
- **lg**: 1280px+ (large desktop)

### Grid Layouts
- **Mobile (xs)**: 1 column
- **Tablet (sm)**: 2 columns
- **Desktop (md)**: 3-4 columns
- **Large (lg)**: 4+ columns

## Accessibility

- Color contrast ratios meet WCAG AA standards
- All buttons have accessible labels
- Form fields have proper labels
- Icons have aria-labels where needed
- Keyboard navigation supported

## Animation Guidelines

### Transitions
- **Fast**: 200ms (hover, clicks)
- **Medium**: 300ms (dialogs, menus)
- **Slow**: 500ms (page transitions)

### Easing
- `ease-in-out` for most animations
- `cubic-bezier(0.4, 0, 0.2, 1)` for Material Design
